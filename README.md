# TK-Design System

Vue 3 versiyonu üzerinde, Design System denemesi yapildi. 
BEM (Block Element Modifier) css metodolojisi, Card bileşeninde kullanıldı.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
